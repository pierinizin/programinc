# Incovia Web - Layout otimizado

## Rodar
```bash
npm install
npm run dev
```

Se o PowerShell bloquear `npm`, use:

```bash
npm.cmd install
npm.cmd run dev
```

## O que já está no pacote
- Layout otimizado da programação
- Tipo de equipe em lista
- Tipo de serviço em lista
- Status com `EXECUTANDO`
- Motivo ocupando largura total
- Horários em grade 2x2
- Exportação:
  - Atual
  - Modelo antigo
  - Pessoas
  - Veículos
  - Histórico

Observação: os exports usam formato `.xls` compatível com Excel para manter estabilidade sem dependências externas.
